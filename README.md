## 기능
regex 로 추출한 문자를 hashMap에 저장

### 예제)
[[1번축사]] 송아지 한마리 탈출<br>
key : 1번축사<br> value : 송아지 한마리 탈출

### 테스트케이스
1. 단일 아이디 입력 시 ->  &nbsp; &nbsp; &nbsp;[[1번축사]] 오늘은 배가 고프다
2. 잘못된 아이디 형식 입력 시 ->  &nbsp; &nbsp; &nbsp;[[1번축사,]] 오늘은 배가 고프다
3. 복수 아이디 입력 시 ->  &nbsp; &nbsp; &nbsp;[[1번축사,2번축사]] 오늘은 배가 고프다
